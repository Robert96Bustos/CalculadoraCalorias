Método de Harris-Benedict

El método de Harris-Benedict es una fórmula utilizada para estimar las necesidades calóricas diarias de una persona, tomando en cuenta su género, peso, altura, edad y nivel de actividad física.

Para hombres: TMB = 88.362 + (13.397 x peso en kg) + (4.799 x altura en cm) - (5.677 x edad en años)

Para mujeres: TMB = 447.593 + (9.247 x peso en kg) + (3.098 x altura en cm) - (4.330 x edad en años)

Después de calcular la TMB, se multiplica por un factor correspondiente al nivel de actividad física para obtener el total de calorías diarias recomendadas.
